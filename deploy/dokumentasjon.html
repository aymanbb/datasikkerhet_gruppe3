Frontend

Som angitt i oppgaven er styling ikke prioritert. HTML er delvis generert med hjelp av KI-tjenesten Google Gemini, det er skrevet minimal CSS - for hånd.

API

Trykk her for å komme til api-spesifikk dokumentasjon.

Det er opprettet et API for interaksjon mellom frontend og databasen. API'et er et POST-basert API som gjør det mulig å utføre enkle operasjoner mot løsningen. Det er tilrettelagt både mot webtjenester og tredjepartsløsninger som applikasjoner.
